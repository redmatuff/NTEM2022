Definição

Função Exponencial:

Dado um número real a (a > 0 e a ≠ 1), denomina‐se função exponencial de base a, a função f:  R → R*+ representada, para todo x real, por

f(x) = a^x

Uma função exponencial é uma função cuja variável está em um expoente e cuja base é sempre maior que zero e diferente de um. 

Essas restrições são necessárias porque aumentar 1 para qualquer número dá 1. Portanto, estaremos lidando com uma função constante, não uma função exponencial. Além disso, a base não pode ser negativa ou igual a zero, pois para alguns expoentes a função não estará definida.

Por exemplo, a base é igual a -3 e o expoente é igual a 1/2. Como não há raiz quadrada de um número negativo no conjunto dos números reais, não haverá gráfico de função para esse valor.

Exemplos de Funções Exponenciais:

f(x) = (\frac{1}{2})^x

f(x) = 2^x

f(x) = 7^x + 1
