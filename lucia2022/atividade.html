Pergunta

Determine o valore reai de m para que a seguinte equação tenha solução: a) sen x = 2m - 7

Respostas

{m ∈ R | 3 ≤ m ≤ 4}

{m ∈ R | m ≤ 4}

{m ∈ R | m ≤ 3}

{m ∈ R | 3 ≥ m ≥ 4}

Pergunta

Determine o valore reai de m para que a seguinte equação tenha solução: b) sen x = 3m - 2

Respostas

{m ∈ R | 1/3 ≤ m }

{m ∈ R | m ≤ 1}

{m ∈ R | 1/3 ≤ m ≤ 1}

{m ∈ R | 1/2 ≤ m ≤ 1}

Pergunta

Determine o valore reai de m para que a seguinte equação tenha solução: c) 4m + sen x = 1

Respostas

{m ∈ R | 1/2 ≤ m ≤ 0}

{m ∈ R | 1 ≤ m ≤ 0}

{m ∈ R | 0 ≤ m ≤ 1/2}

{m ∈ R | -1 ≤ m ≤ 0}
